# @claudiu-ceia/astkit-core

Shared core utilities for astkit packages (template matching, file scanning, boundary checks, and tracing helpers).

This package is primarily an internal dependency of:

- `@claudiu-ceia/spatch`
- `@claudiu-ceia/sgrep`
- `@claudiu-ceia/nav`

Public APIs are documented in package-level READMEs that consume this core layer.
